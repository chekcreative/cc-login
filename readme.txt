== Description ==

This plugin is intended for Chek Creative-hosted WordPress sites. We developed this small plugin to secure the login process, and enable direct link to our site.

== Installation ==

1. Install plugin via Chek Creative ([hello@chekcreative.com](mailto:hello@chekcreative.com)) support.
